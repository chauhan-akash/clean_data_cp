# clean_data_cp

in the run_analysis.R code following steps are performed:

1. read the test data set 
2. assign the variable names to test data set 
3. keep only mean and std deviation varibales of test
4. append subject code and activity to test
5. append all of the above info to create the final test
6. repeat steps 1-5 for training dataset
7. append train and test data
8. summarize train and test data
9. output as txt file
